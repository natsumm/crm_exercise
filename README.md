1, 首页功能, 跳转登录页面
   用户访问项目首页，首先进入登录页面;
2, 用户登录
   用户在登录页面,输入用户名和密码,点击"登录"按钮或者回车,完成用户登录的功能.
	*用户名和密码不能为空
	*用户名或者密码错误,用户已过期,用户状态被锁定,ip受限 都不能登录成功
	*登录成功之后,所有业务页面显示当前用户的名称
	*实现10天记住密码
	*登录成功之后,跳转到业务主页面
	*登录失败,页面不跳转,提示信息
